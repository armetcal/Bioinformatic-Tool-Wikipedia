Metagenomics Workshop (Dec. 2024)

This webpage contains a series of R scripts that run a suite of the most common gut microbiome analyses using a publicly available dataset. Most tools can be run on both taxonomy and functions, but only taxonomy is used in these tutorials.

The dataset used was published by Wallen et. al (2022) and the annotated data are available for download at https://zenodo.org/records/7246185.

Download the following scripts. 0 and 1 must be run first, and the others can be run in any order.

0. Install necessary packages.R

1. Create phyloseq object.Rmd

2. Alpha Diversity Tutorial.Rmd

3. Beta Diversity Tutorial.Rmd

4. Taxonomy Bar Plot.Rmd

5. Core Microbiome.Rmd

6. Differential Abundance with ANCOM-BC Tutorial.Rmd
