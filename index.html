About

This repository is designed to act as a reproducible guide for running various bioinformatic tools, particularly those related to DNA/protein/RNA/metabolite quantification and similar comparisons

Unless otherwise specified, the following tutorials assume that you are analyzing metagenomic data that are available on the NCBI database.

All tutorials involving unannotated data are adapted for use with Compute Canada resources.
